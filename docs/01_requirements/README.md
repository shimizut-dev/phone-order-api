# 01_requirements

要件整理に関する資料を管理します。

---

## 目的

- システムの対象範囲を明確にする
- 利用者・関係者を明確にする
- 業務ルールを整理する
- 上位設計の前提をそろえる

---

## 管理資料

- `scope.md`
    - 対象範囲
- `actors.md`
    - 利用者・関係者
- `business-rules.md`
    - 業務ルール
- `system-overview.md`
    - システム概要

---

## 更新ルール

- 要件の追加・変更時に更新する
- ドメインや API に影響する変更は関連資料も追従する
- 実装詳細は本ディレクトリではなく下位資料へ記載する
