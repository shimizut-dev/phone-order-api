# actors.md

## 1. 目的

本ドキュメントは `phone-order-api` における利用者（Actor）を定義する。

---

## 2. Actor一覧

### 2.1 オペレーター

注文を登録する利用者。

操作

- 注文作成
- 注文参照
- 注文キャンセル
- 配送作成

### 2.2 管理者

注文情報を管理する利用者。

操作

- 注文参照
- 配送参照

---

## 3. 将来Actor（MVP対象外）

### 3.1 審査担当

- 審査処理

### 3.2 出荷担当

- 出荷処理

---

## 4. Actorの役割

| Actor | 説明 |
|-------|------|
| オペレーター | 注文登録、注文キャンセル、配送作成 |
| 管理者 | 注文参照、配送参照 |
| 審査担当 | 審査処理（将来） |
| 出荷担当 | 出荷処理（将来） |

---

## 5. 補足

- MVPでは、配送作成はオペレーターが実施する
- 管理者は参照系操作を中心とする
