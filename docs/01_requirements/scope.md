# scope.md

## 1. 目的

本ドキュメントは `phone-order-api` のスコープを定義する。  
MVPで実装対象とする範囲および対象外を明確化することを目的とする。

---

## 2. スコープ内（MVP対象）

### 2.1 注文管理

- 注文作成
- 注文参照
- 注文キャンセル

### 2.2 注文明細

- 回線注文
- 移動機注文
- アクセサリ注文
- SIM単体申込

### 2.3 注文関係者

- 注文者
- 契約者
- 利用者
- 請求者

### 2.4 配送

- 配送作成
- 配送参照
- 分納

---

## 3. スコープ外（MVP対象外）

### 3.1 注文

- 回線のみ申込
- 注文変更
- 注文復旧

### 3.2 業務処理

- 審査
- 支払
- キャンペーン
- 返品

### 3.3 在庫

- 在庫管理
- 在庫引当

### 3.4 外部連携

- 外部配送会社連携
- 外部在庫連携
- 外部課金連携

---

## 4. 前提条件

- MVPはシンプルな注文管理を目的とする
- 業務フローは最小限とする
- 将来拡張を前提とした設計とする
