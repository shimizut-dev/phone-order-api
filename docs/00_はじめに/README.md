# 00_はじめに

本ディレクトリは `docs/` 全体の入口です。  
`docs/` の全体像、カテゴリ構成、作成方針、テンプレートへの導線を管理します。

---

## 概要

- `docs/` 全体の入口を明確にする
- 初めて参照する人が読む順序を揃える
- ドキュメントの全体方針と配置先を分かりやすくする
- ドキュメント作成時のテンプレートを共通化する

---

## 管理資料

- `ドキュメントガイドライン.md`
  - `docs/` 全体の方針、カテゴリ一覧、命名、配置、テンプレート利用ルール
- `テンプレート/汎用テンプレート.md`
  - 通常の設計ドキュメントやルールドキュメントに使う共通テンプレート
- `テンプレート/READMEテンプレート.md`
  - 各ディレクトリ用 `README.md` の共通テンプレート
