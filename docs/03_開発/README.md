# 03_開発

本ディレクトリは `phone-order-api` の開発ガイドを管理します。  
開発フロー、規約、環境設定など日々の開発運用に関する資料を整理します。

---

## 概要

- 日々の開発手順を揃える
- 実装時の判断基準を共有する
- ローカル開発やレビューの前提を統一する

---

## 管理資料

- `開発フロー.md`
  - 日常の開発手順とレビュー前提を整理する
- `Git運用/`
  - ブランチ命名とコミットメッセージを配置する
- `コーディング規約/`
  - 実装命名、値オブジェクト、JPA、ログ、Javadoc などの規約を配置する
- `開発環境/`
  - ローカル実行と環境構成の資料を配置する
- `IDE設定/`
  - IntelliJ 関連資料と利用プラグイン前提を配置する
