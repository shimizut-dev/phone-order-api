# A5:SQL DDL自動生成ガイド

本資料は A5:SQL Mk-2 を使用した DDL 自動生成の設定値と、
自動生成後に手動修正するルールを定義する。

---

# 1. 方針

DDLは以下の方針で作成する

- 基本は A5:SQL で自動生成
- 命名規則に関する部分のみ手動修正
- 可読性を優先する

---

# 2. A5:SQL DDL自動生成設定値

## 基本設定

| 項目                               | 設定         |
|----------------------------------|------------|
| RDBMS種別                          | PostgreSQL |
| Create table にコメントを生成する          | ON         |
| テーブルオプションを有効にする                  | ON         |
| リレーションシップから外部キー制約を作成する           | ON         |
| Drop tableやCreate tableでデータを保持する | OFF        |
| 識別子をダブルクォートで囲む                   | OFF        |

---

# 3. 制約名設定

## 主キー制約名

```
pk_%0:s
```

## 外部キー名

```
fk_%0:s_%1:d
```

## インデックス名

```
idx_%0:s_%1:d
```

---

# 4. 手動修正ルール

## 外部キー

自動生成

```
fk_order_lines_1
```

修正後

```
fk_order_lines_order_id
```

---

# 5. まとめ

- DDLは自動生成
- 制約名のみ手動修正
- 可読性優先
