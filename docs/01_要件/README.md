# 01_要件

本ディレクトリは `phone-order-api` の要件を管理します。  
現行実装と API 契約から確認できる範囲の要求事項を整理し、上位前提を共有します。

---

## 概要

- システムの対象範囲を明確にする
- 利用者と利用場面を整理する
- 業務上の制約を整理する
- 設計と実装の前提となる要件を共有する

---

## 管理資料

- `スコープ.md`
  - 対象業務と対象外を整理する
- `関係者.md`
  - 想定する利用者と関係者を整理する
- `業務ルール.md`
  - 注文管理に関する業務上の制約を整理する
- `システム概要.md`
  - API とデータ管理の概要を整理する
