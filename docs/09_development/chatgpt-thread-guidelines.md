# 🤖 ChatGPTスレ運用ガイド

## 🎯 目的

本プロジェクトでは ChatGPT を活用して開発を進めるため、  
スレッド（スレ）を分割して管理する。

スレを分割することで以下のメリットがある

- 可読性向上
- 動作の軽量化
- 情報整理
- 開発効率向上

---

# 🧭 基本方針

開発フェーズごとにスレを作成する。

---

# 📂 スレ一覧

| スレ名       | 主な内容                                   |
|-----------|----------------------------------------|
| 📌要件スレ    | 機能要件 / 非機能要件 / スコープ / レビュー             |
| 🧩設計スレ    | ドメイン / API / DB / アーキテクチャ / レビュー       |
| 🔧実装スレ    | 実装方針 / コード設計 / レビュー                    |
| 📏開発ルールスレ | Git / PR / Issue / AI運用 / 開発ルール / レビュー |
| 🔍調査・雑談スレ | 技術調査 / 方針相談                            |

---

# 📌 AIの利用方針

AIは以下のスレで利用する

| スレ名       | 内容          |
|-----------|-------------|
| 📌要件スレ    | AIで要件を作成    |
| 🧩設計スレ    | AIで設計を作成    |
| 🔧実装スレ    | AIで実装を作成    |
| 📏開発ルールスレ | AIで開発ルールを作成 |
| 🔍調査・雑談スレ | AIで技術調査     |

---

# 🧱 成果物作成フロー

スレ内では以下のフローで成果物を作成する

```text
📝 作成 → 👀 レビュー → 🔧 修正 → (🔁繰り返し) → ✅ 完成
```

---

# 📋 工程一覧

| 工程      | 作業者           | 作業内容             | 成果物    |
|---------|---------------|------------------|--------|
| 📝 作成   | ユーザ           | 作成対象の指示 / 方向性提示  | 作成依頼   |
| 📝 作成   | ChatGPT       | ドラフト作成 / 不明点整理   | ドラフト   |
| 👀 レビュー | ユーザ           | 内容確認 / 修正依頼 / 指摘 | レビュー指摘 |
| 👀 レビュー | ChatGPT       | 抜け漏れチェック / 改善提案  | レビュー結果 |
| 🔧 修正   | ユーザ           | 修正方針提示 / 修正指示    | 修正依頼   |
| 🔧 修正   | ChatGPT       | 修正対応 / 再作成       | 修正版    |
| 🔁 繰り返し | ユーザ / ChatGPT | レビューと修正を繰り返す     | 更新版    |
| ✅ 完成    | ユーザ           | 最終確認 / 完成判断      | 完成版    |

---

# 📌 運用ルール

- 作成・レビュー・修正は同一スレで実施する
- レビューと修正は必要に応じて繰り返す
- 完成したら資料として確定する

---

# ✨ 効果

この運用により

- ChatGPT動作軽量化
- 開発効率向上
- 可読性向上
- 開発プロセスの明確化

を実現する。
