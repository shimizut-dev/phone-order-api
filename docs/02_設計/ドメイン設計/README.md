# ドメイン設計

本ディレクトリは `phone-order-api` のドメイン設計資料を管理します。  
注文を中心とした用語、ルール、モデルを整理します。

---

## 概要

- 業務概念を共通の言葉で扱えるようにする
- ドメインルールとモデルの関係を整理する
- 実装と業務ルールの対応を明確にする

---

## 管理資料

- `用語集.md`
  - ドメインで扱う主要な用語を整理する
- `注文集約.md`
  - 注文集約の構造と責務を整理する
- `注文明細生成ルール.md`
  - 注文明細に関する現状と今後の整理方針を記載する
- `ルール/`
  - ドメインルールと集約、エンティティ、値オブジェクト、ドメインサービス、リポジトリ、ファクトリの整理を配置する
- `モデル/`
  - モデル図とその元データを配置する
