# 02_設計

本ディレクトリは `phone-order-api` の設計資料を管理します。  
ユースケース、ドメイン設計、API 設計、アーキテクチャ設計、データベース設計を整理します。

---

## 概要

- 設計上の責務と構造を分野ごとに整理する
- 要件から実装へ落とすための中間資料を管理する
- 主要な設計判断の前提を参照しやすくする

---

## 管理資料

- `ユースケース/`
  - 利用シナリオとユースケース一覧を配置する
- `ドメイン設計/`
  - 用語、集約、ドメインルール、モデルを配置する
- `API設計/`
  - OpenAPI 仕様、API 設計ルール、エラーレスポンスを配置する
- `アーキテクチャ設計/`
  - レイヤ、依存、パッケージ構成、図を配置する
- `データベース設計/`
  - DB 概要、採番、命名、ER 図を配置する
