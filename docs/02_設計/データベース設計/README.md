# データベース設計

本ディレクトリは `phone-order-api` のデータベース設計資料を管理します。  
テーブル構造、採番、命名、ER 図関連資料を整理します。

---

## 概要

- 現行 DB 設計の前提とテーブル構造を共有する
- ID 採番と命名ルールを明確にする
- ER 図と DDL 補助資料を管理する

---

## 管理資料

- `データベース概要.md`
  - DB 製品、監査カラム、論理削除カラム、`orders` テーブル構成を整理する
- `ID採番方針.md`
  - `OrderId` と `order_code` の採番方針を整理する
- `命名ルール.md`
  - テーブル、カラム、制約、インデックスの命名ルールを整理する
- `ER図/`
  - ER 図と元データを配置する
- `補助資料/`
  - A5:SQL Mk-2 利用時の手順と ER 図作成時の補助資料を配置する
